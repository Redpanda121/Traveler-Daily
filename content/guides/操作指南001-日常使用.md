> 这是一篇写给未来自己的操作记录。

## 启用本地预览

进入博客目录：
```powershell
cd "E:\文件\【旅者手札】\My-Blog"
```

启动Hugo：
```powershell
hugo server -D
```

浏览器打开：
```text
http://127.0.0.1:1313/
```

停止预览：
```text
Ctrl + C
```
## 新建文件

新建一篇日记：
```powershell
hugo new content/posts/冒险日志-{日期}.md
```

新建操作指南：
```powershell
hugo new content/posts/操作指南{编号}-{标题}.md
```

## Git设置

查看状态：
```powershell
git status
```

添加修改：
```powershell
git add .
```

提交存档：
```powershell
git commit -m "add new post"
```

查看提交记录：
```powershell
git log --oneline
```

## 常见问题

### localhost 打不开

优先打开：
```text
http://127.0.0.1:1313/
```

如果端口有问题，换端口：
```powershell
hugo server -D --bind 127.0.0.1 --port 1314 --baseURL http://127.0.0.1:1314/
```

### git add 出现 LF/CRLF warning
这是 Windows 换行符提示，一般可以忽略。

### git commit 提示 Author identity unknown

设置一次身份：
```powershell
git config --global user.name "用户名" git config --global user.email "你的邮箱"
```
然后重新提交：
```powershell
git commit -m "init Blog"
```
